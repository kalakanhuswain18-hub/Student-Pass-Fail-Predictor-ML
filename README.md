# 🎓 Student Pass/Fail Predictor (ML)

This is a beginner-friendly Machine Learning project that predicts whether a student will PASS or FAIL based on:

- 📚 Study Hours  
- 🔁 Revision Time  

---

## 🚀 Tech Stack
- Python 🐍
- Pandas 📊
- Scikit-learn 🤖

---

## 🧠 Model Used
- Logistic Regression (Classification)

---

## 📊 How It Works
The model is trained on a dataset containing study hours and revision time.

It learns patterns like:
- Less study → FAIL ❌  
- More study + revision → PASS ✅
